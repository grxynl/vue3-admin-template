# vue3-admin-template

vue3+TypeScript+pinia 后台管理系统模板

***本框架主要的技术栈：***

> Vue 3.2+  
> vite 4.2  
> typescript 4.9.3
> pinia 2.1.4
> element-plus 2.1.0  
> axios 1.3.4  
> less 4.1.3  
> echarts 5.4.2  
> mockjs 1.1.0
